# Installation & Usage
Clone repository:
```
git clone <this_repo>
```
Client-side (PORT:5173)
```
cd client
npm i
npm run dev
```
Server-side (PORT:5000)
```
cd server
npm i
nodemon server
```
Create .env and add mongoDB connection:
```
mongodb_url =
 ```

You can get the data from StaticData.js
